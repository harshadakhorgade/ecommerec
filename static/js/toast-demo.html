Usage in Your Django Project

The toast system automatically converts Django messages to toast notifications. Your existing code will work without changes:

# In your Django views (no changes needed) messages.success(request, 'Product added to cart!') messages.error(request, 'Invalid credentials.') messages.warning(request, 'Session expired.') messages.info(request, 'Profile updated successfully.')

Manual Toast Creation

You can also create toasts manually in JavaScript:

// Show different types of toasts showSuccess('Operation completed successfully!'); showError('Something went wrong!'); showWarning('Please be careful!'); showInfo('Here is some information!'); // Or use the generic function showToast('Custom message', 'success', 3000); // 3 second duration
